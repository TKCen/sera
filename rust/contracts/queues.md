# SERA Queue & Channel Inventory

## pg-boss Queues

| Queue Name              | Owner Service           | Purpose                                 |
| ----------------------- | ----------------------- | --------------------------------------- |
| `agent-schedule`        | ScheduleService         | Main worker queue for schedule dispatch |
| `{schedule-uuid}`       | ScheduleService         | Per-schedule cron queue (UUID as name)  |
| `notification.dispatch` | NotificationService     | Outbound notification delivery          |
| `memory.compaction`     | MemoryCompactionService | Chat history compaction jobs            |

### Queue Semantics

- All queues use pg-boss v9+ which requires `createQueue()` before `schedule()` or `send()`
- Per-schedule queues use the schedule's UUID as the queue name
- `agent-schedule` is the single worker queue created in `ScheduleService.start()`
- Job payloads are JSONB — plain strings must be wrapped as `{"prompt": "..."}`

## Centrifugo Channel Prefixes

| Prefix                        | Purpose                       | Publisher                    |
| ----------------------------- | ----------------------------- | ---------------------------- |
| `internal:agent:{instanceId}` | Agent-to-agent messaging      | IntercomService              |
| `agent:{instanceId}`          | Agent status/lifecycle events | Orchestrator, SandboxManager |
| `agent:{instanceId}:thoughts` | Real-time thought streaming   | BaseAgent reasoning loop     |
| `agent:{instanceId}:tokens`   | Token-by-token LLM streaming  | BaseAgent, WorkerAgent       |
| `broadcast:system`            | System-wide announcements     | Orchestrator                 |
| `broadcast:circle:{circleId}` | Circle-scoped broadcasts      | CircleRegistry               |

### Channel Conventions

- `internal:*` channels are for server-to-server communication
- `agent:*` channels carry real-time data to web UI subscribers
- `broadcast:*` channels are fan-out to all subscribers
- JWT tokens for channel subscription generated by `IntercomService.generateConnectionToken()`
- Centrifugo HMAC secret must match `CENTRIFUGO_TOKEN_SECRET` env var
